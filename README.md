# Hertaa-Playground: A Simple WhatsApp Bot Base (NekoBot-Based)

[![Thumb](https://files.catbox.moe/cfwa8y.png)]

Hertaa-Playground is a **simplified version of [NekoBot](https://github.com/YOUR_NEKOBOT_LINK_HERE)**

**Getting Started:**

1.  **Clone the repository:**

    ```bash
    git clone [https://github.com/YOUR_USERNAME/Hertaa-Playground.git](https://www.google.com/search?q=https://github.com/YOUR_USERNAME/Hertaa-Playground.git)
    ```

2.  **Install Dependencies:**

    ```bash
    cd Hertaa-Playground
    npm install
    ```

3.  **Configuration:**
    * Edit `settings.js` and set your desired `customPairCode`

4.  **Run the Bot:**

    ```bash
    npm start
    ```

5.  **Enter WhatsApp Number and Pairing Code:**
    Follow the instructions provided by the bot.
